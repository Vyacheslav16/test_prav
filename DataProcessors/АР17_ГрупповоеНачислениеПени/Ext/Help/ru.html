Обработка "Групповое начисление пени"

 

Обработка предназначена для группового формирования документов Начисление пени на выбранную дату.

На закладке Отбор данных имеется возможность указания условий, по которым будет происходить отбор данных для начисления пени. Отбор можно производить по следующим полям:

■      Услуга;

■      Контрагент;

■      Договор контрагента;

■      Объект аренды.

Если отбор не указан, то начисления пени будут формироваться по всем арендаторам, услугам, объектам аренды.

С помощью признаков Учитывать НДС и Сумма включает НДС регулируется способ учета НДС по услугам в формируемых документах.

Для формирования документов необходимо нажать кнопку Сформировать документы.

Все сформированные документы отобразятся на закладке Сформированные документы, где их можно просмотреть, распечатать (с предварительным просмотром или сразу на принтер), провести либо удалить.

На основании сформированных документов можно оформить документы Счет-фактура выданный. При нажатии на кнопку Выставление СФ откроется групповая обработка Групповое выставление счетов-фактур с установленными настройками текущей обработки.
