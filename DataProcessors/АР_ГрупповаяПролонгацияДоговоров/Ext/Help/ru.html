Обработка "Групповая пролонгация договоров аренды"

 

Обработка предназначена для формирования пакета документов «Дополнительное соглашение» на продление сроков аренды по указанную дату или на определенный срок.

 

На форме обработки при помощи установленного отбора и кнопок «Заполнить по отбору», «Добавить по отбору» необходимо заполнить список арендаторов. Пролонгация договоров будет осуществлена только для арендаторов из этого списка.

 

Непосредственное формирование документов «Дополнительное соглашение» производится после нажатия кнопки «Сформировать документы».
